# Credit Risk Scoring using IBM SPSS Modeler

## Objective
To predict whether a customer is a good or bad credit risk using machine learning models.

## Dataset
- German Credit Dataset
- 1000 records
- Features include age, credit history, account status, etc.
- Target variable: good/bad

## Methodology
1. Data loaded using Excel node
2. Type node used to define target variable
3. Data split into training (70%) and testing (30%)
4. Models built:
   - Logistic Regression
   - Decision Tree (C&R Tree)
5. Predictions generated using model nuggets

## Results
- Logistic Regression provided reliable predictions
- Decision Tree performed slightly better and was more interpretable

## Conclusion
Decision Tree model is more suitable for credit risk prediction due to better interpretability and performance.

## Tools Used
- IBM SPSS Modeler
